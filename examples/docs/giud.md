# 介绍

----

`lulu-UI` 是一款基于 `Vue.js v2.5.2` 的前端 UI 组件库，主要用于是个人学习，比较喜欢简洁的样式，参考了[lulu](https://github.com/VV-UI/VV-UI/pulls)


## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- [Electron](http://electron.atom.io/)
- [NW.js](http://nwjs.io)
