
# 介绍

----

`lulu-UI` 是一款基于 `Vue.js v2.5.2` 的前端 UI 组件库，个人业余时间做的，样式模仿了[lulu](https://l-ui.com/)，构建方式以及组件的封装学习了多个库iview，antd，element-ui，vv-ui等


## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提供友好的 API，可灵活的使用组件

## 浏览器支持

- 现代浏览器和 IE9 及以上
- [Electron](http://electron.atom.io/)
- [NW.js](http://nwjs.io)
